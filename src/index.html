Retail Online Banking

Ajman Bank will never ask you for an SMS Security Code in order to login to Online Banking. If you have received an SMS with the SMS Security Code that you have not requested for, please do not act on it and immediately contact us on 800 22 if you are in the UAE and on +971 6 701 8484 if you are outside UAE. For more information on how to use Online Banking safely please visit our online Banking Security Tips Section.
